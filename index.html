Installation Steps

  1. Connect your ESP32 device to your computer via USB
  2. Click "Connect & Install Firmware"
  3. Select your device from the popup
  4. Wait for the installation to complete

Firmware Lifecycle Management

How It Works

  1. Add YAML File: Create a firmware configuration in the iot-firmware-src repository
  2. Automatic Build: CI/CD pipeline detects changes and builds firmware
  3. Auto-Deploy: Firmware is automatically deployed to this web interface
  4. Interface Update: This page updates to show available firmware options

Troubleshooting

No Firmware Available

Add a YAML configuration file to the iot-firmware-src repository to trigger firmware building.

Device Not Detected

Make sure your device is connected via USB and the correct drivers are installed.

Installation Failed

Try putting your device in download mode by holding the BOOT button while connecting.

Connection Issues

Ensure no other programs are using the serial port. Close Arduino IDE or other serial monitors.
