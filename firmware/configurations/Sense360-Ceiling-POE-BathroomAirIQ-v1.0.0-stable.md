# Sense360 Ceiling Mount POE with Bathroom AirIQ v1.0.0 (Stable)

## Configuration Details
- **Mounting Type**: Ceiling
- **Power Option**: POE Module
- **Expansion Modules**: Bathroom AirIQ Base
- **Included Sensors**: SGP41, SCD41, Humidity, Temperature
- **Chip Family**: ESP32-S3
- **Version**: v1.0.0
- **Channel**: stable
- **Release Date**: 2025-12-10

## Description
Stable release firmware for ceiling-mounted Sense360 nodes powered over Ethernet with the Bathroom AirIQ module. Optimized for bathroom environments with enhanced humidity and air quality monitoring.

## Hardware Requirements
- ESP32-S3 WROOM Core Module
- POE Module for power and data
- Bathroom AirIQ Module with SGP41, SCD41, humidity, and temperature sensors
- Ceiling mounting hardware

## Features
- Production-ready VOC and gas monitoring optimized for bathroom environments
- Enhanced humidity monitoring for moisture detection
- Stable POE power management profiles
- Improv Wi-Fi provisioning compatible with ESP Web Tools
- MQTT telemetry formatted for Home Assistant integrations

## Notes
- Bathroom AirIQ is only available for ceiling mount installations
- Cannot be used simultaneously with standard AirIQ module
